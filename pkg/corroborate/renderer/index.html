overview
Validated GPU cluster configurations — independently corroborated
Find a GPU cluster configuration that independent sources have validated. Each recipe's status is corroborated across sources — disagreement is surfaced, never averaged.
How to read this dashboard

Pick a cloud service (eks/gke/aks/oke/lke), then an accelerator-os, then a recipe (intent[-platform]). A recipe opens its cross-source corroboration grid; clicking a source column opens that source's builds-over-time history.

Consensus states
Dashboard eks · recipes × phases
Each row is a recipe; each cell is the phase consensus rolled up across all sources (worst-first: CONTESTED > FAILING > UNTESTED > SINGLE > CONFIRMED). Select a recipe to open its corroboration matrix.
